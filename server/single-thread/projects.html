Shopping List

Github Repo
  • An android app made to keep track of my shopping list.
  • Used android studio and the android api, written in Java.
  • Made use of the RecyclerView provided by android to keep track of the updating list.
  • Published on the android Play Store and can be downloaded here.

Maze Runner

  • A Java project which models a maze, and successfully navigates the maze from entrance to exit.
  • Has the ability to read and write mazes to file.
  • Uses a breadth first search with recursion on a queue of stacks.

Railway Network

  • A JavaScript project that models a railway network, and finds the most efficient route between two stops.
  • Implements an adjacency list to map the railway as a graph.
  • Contains a module of several useful functions used to interact with the railway object.
  • Uses a depth first search to find detailed routes between two stations.

Contact List

  • a Java project which models a basic database of contacts and can perform basic set operations on.
  • Performs operations such as intersection, union, and subtraction on two databases.
  • Can perform a search for a contact based on any piece of information given (phone number, email, first/last name, etc)

Black Jack

  • A game of blackjack programmed in Java played on a console against a dealer.
  • Shuffles and keeps track of a standard deck of 52 cards.
  • Implements the Fisher-Yates shuffling algorithm.

OrderedSet and OrderedLinked

  • A set of Java ADT's which model a set of unique geniric items which is automatically sorted as items are added.
  • OrderedSet implements an array which is resized as necessary.
  • OrderedLinked implements a linked list.
  • Created a tester to test all uses of both ADT's

Cookie Clicker Fullscreen Shortcut

Github Repo
  • A mod for the steam version of Cookie Clicker which adds the functionality to toggle Fullscreen with f11.
  • Implemented the Cookie Clicker modding api.
  • Published on the steam workshop.
